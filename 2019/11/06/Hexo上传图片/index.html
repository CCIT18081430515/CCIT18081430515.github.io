Hexo上传图片

1.打开博客配置文件_config.yml，将post_asset_folder这个选项设置为true
2.在Hexo安装目录下执行:npm install hexo-asset-image –save，这是下载安装一个可以上传本地图片的插件
3.等待一小段时间后，再运行hexo n “xxxx”来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹
4.最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片： 注意，此处的图片路径必须使用相对路径
5。执行hexo g,检查生成的页面中图片的src地址。此时生成页面中图片src地址应该与页面的相对路径一致（具体路径取决于页面路径格式设置）

注：以上方法是Hexo2.0.0版本，如果是Hexo1.0.0版本，需要在博客根目录下的package.ison文件，将kexo-asset-image的版本号改成0.0.1。然后再Hexo目录下执行npm install
